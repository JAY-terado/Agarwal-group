AGARWAL GROUP — WEBSITE PACKAGE
================================

HOW TO USE
----------
Keep ALL files in this folder together (same directory) and upload them to your
web hosting's public root. The homepage is index.html. All links between pages
and all images are relative, so the site works as-is once the files sit together.

PAGES
-----
index.html               Homepage
agarwal-paramount.html   Project page — Agarwal Paramount
agarwal-exotica.html     Project page — Agarwal Exotica
agarwal-solitaire.html   Project page — Agarwal Solitaire
agarwal-lifestyle.html   Project page — Agarwal Lifestyle
blog.html                Blog / Property Insights
channel-partner.html     Channel Partner

IMAGES / MEDIA (must stay in the same folder)
---------------------------------------------
hero.mp4, hero-poster.jpg ............ homepage hero video + fallback poster
agarwal-*-hero.jpg ................... the 4 project hero photos
gallery-*.jpg ........................ shared project gallery photos
testimonial-1..6.jpg ................. homepage testimonial photos
blog-*.jpg ........................... blog post + homepage insights photos

ONE FILE YOU STILL NEED TO ADD
------------------------------
Agarwal_Group_Logo.png  — the logo used in the top nav and footer of every page.
Drop your logo PNG into this folder with exactly that filename and it will appear
everywhere automatically. (Until then, the logo shows as a broken image.)

EDITING NOTES
-------------
- The 4 project pages share one structure; edit the text between the <!-- EDIT -->
  comments to create new project pages (duplicate a file, rename, change content).
- To use a project hero photo, the CSS already points to agarwal-<name>-hero.jpg.
- Testimonial cards show a play button by design; they are images (not videos).
